# Markdown入门

（代码与效果演示文档）

我认为，有记笔记习惯的，最好都学一下 markdown 。

不管是课堂记笔记，还是个人写作，都可以使用，方便快捷。

特别是计算机类专业的，使用专门的代码块格式，还可以生成对应的语法渲染，可读性很高。即使你电脑不在身边，只有手机，也可以快速排版，后续复制到电脑端即可，非常方便。

# 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

或者


二级标题
---

二级标题
===
```

二级标题
---

二级标题
===

示例2~6级：


## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 基础常用

## 加粗斜体
代码：

```markdown
**加粗1**
*斜体1*
__加粗2__
_斜体2_

***加粗斜体1***
___加粗斜体2___
```

效果：
**加粗1**
*斜体1*
__加粗2__
_斜体2_
***加粗斜体1***
___加粗斜体2___


## 高亮
```markdown
==高亮==
<mark>高亮</mark>
```

注意：部分markdown编辑器不支持`==高亮==`，需要使用`<mark>高亮</mark>`才行。

效果：
==高亮==
<mark>高亮</mark>


## 分割线

```markdown
---

***

```

效果：


---

***


## 列表
### 无序列表

```markdown
- 减号1
- 减号2

* 星号1
* 星号2

+ 加号1
+ 加号2

```

效果：

- 减号1
- 减号2

* 星号1
* 星号2

+ 加号1
+ 加号2

### 有序列表
```markdown
1. 列表序号 1
2. 列表序号 2
3. 列表序号 3
4. 列表序号 4
```
或者

```markdown
1. 列表序号 1
1. 列表序号 2
1. 列表序号 3
1. 列表序号 4
1. 列表序号 5
```


效果：

1. 列表序号 1
2. 列表序号 2
3. 列表序号 3
4. 列表序号 4

---


1. 列表序号 1
1. 列表序号 2
1. 列表序号 3
1. 列表序号 4
1. 列表序号 5


## 代码块
普通写作不一定常用，开发者更常用。

### 行内代码

```markdown
`假设这是一行代码`
```

效果：
`假设这是一行代码`

### 多行代码
多行代码可以设置编程语言：

````markdown
```
假设这是多行代码
第二行
第三行
```
````

### 示例

#### 比如记录Python笔记

```python
'''
这是Python
的多行注释
'''
print("666")  #这是单行注释，可以写在代码后面
```
